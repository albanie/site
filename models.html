SSD Detector for matconvnet

We have created a matconvnet implementation of the Single Shot Multibox Detector. The SSD detector, developed by Wei Liu et al. and published at ECCV 2016) is a powerful framework for object detection which has demonstrated impressive accuracy at high frame rates.

Several pretrained caffe models have been made publicly avaiable by Wei Liu (the code and the links to the models can be found here). These have been converted for use in matconvnet and can be downloaded from the links below. In addition, sample models which have been trained with the matconvnet implementation directly are also available for download.
